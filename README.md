## Vision
無驗證治理容易失序。PeaceDAO 以 **$世界和平** 持倉門檻與冷卻機制，搭配公開透明的捐贈金流（全部經由 PeaceFund），在「開放」與「守門」之間取得平衡。

## Governance Highlights
- 門檻（例）：發言 ≥ 100、投票 ≥ 200,000、提案 ≥ 1,000,000 $世界和平  
- 反女巫：提案冷卻、重大金額提案需加強審查  
- 透明金流：所有 BNB 經 **PeaceFund**，驗證者/社群管理者分潤可追溯
